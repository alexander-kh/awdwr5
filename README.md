* Versions used:
  * Ruby: 2.3.0
  * Rails: 5.0.1

* Database setup:
  * Run migrations:
      ```
      bin/rails db:migrate
      ```

  * Add initial data:
      ```
      bin/rails db:seed
      ```

* To run the test suite:
   ```
   bin/rails test
   ```
